# WaitLess

## Problem
Patients waste time waiting in long hospital queues.

## Solution
WaitLess is a real-time queue management system where patients can:
- Scan QR
- See live position
- Get estimated wait time

## Features
- Login/Register UI
- Hospital selection
- Appointment booking
- Token system (prototype)

## Known Issues
- Queue mismatch bug
- Dashboard not completed
- Backend incomplete

## Tech Stack
- React (Frontend)
- Node.js (if used)

## How to Run
1. Open frontend folder
2. Run: npm install
3. Run: npm start
